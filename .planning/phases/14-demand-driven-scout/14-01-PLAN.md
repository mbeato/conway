- Export `async function getAutocompleteSuggestions(query: string): Promise<string[]>`
- GET `https://suggestqueries.google.com/complete/search?client=firefox&q=${encodeURIComponent(query)}&hl=en`
- Use `AbortSignal.timeout(5_000)`
- Parse response format: `[query, [suggestion1, suggestion2, ...]]` — return `data[1]` as string array
- Return `[]` on any error

**rapidapi.ts** (per DEMAND-02):
